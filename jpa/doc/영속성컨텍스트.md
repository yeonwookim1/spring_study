  - 객체를 생성만한 상태

  - ```java
    Member member = new Member();
    member.setId(27L);
    ```

  ​

- 영속

  - 영속성 컨텍스트에 관리되는 상태

  - ```java
    Member member = new Member();
    member.setId(27L);
    EntityManager em = enf.createEntityManager();
    em.getTransaction().begin();

    //객체를 저자한 영속 상태
    em.persisit(member);
    ```
